# Vaults & Copy Trading Complete Guide

## What Are Vaults?

User-created trading strategies that others can invest in, enabling copy trading and strategy monetization.

### Vault Ecosystem
```
Vault Creator → Creates strategy → Trades actively
     ↓
Vault Investors → Deposit USDC → Copy trades automatically
     ↓
Performance Fee → Creator earns → Investors profit
```

### Types of Vaults
```
1. Manual Trading Vaults (human traders)
2. Algorithmic Vaults (automated bots)
3. Hybrid Vaults (human + automation)
4. Index Vaults (basket strategies)
```

## How Vaults Work

### For Vault Creators

#### Creating a Vault
```
Requirements:
- Minimum $1,000 initial deposit
- Trading history (recommended)
- Clear strategy description
- Performance fee setting (0-50%)

Process:
1. Go to Vaults section
2. Click "Create Vault"
3. Set parameters
4. Deposit initial capital
5. Start trading
```

#### Vault Parameters
```
Name: Your vault identity
Description: Strategy explanation
Performance Fee: 0-50% of profits
Management Fee: 0% (not available yet)
Capacity: Maximum deposits accepted
Min Investment: Minimum per investor
```

#### Fee Structure
```
Performance Fee Only Model:
- No management fee
- Only earn on profits
- Paid on high-water mark
- Calculated daily

Example:
20% performance fee
Vault makes $10,000 profit
Creator earns $2,000
Investors keep $8,000
```

## Evaluating Vault Strategies

### Strategy Types

#### Trend Following
```
Characteristics:
- Rides market trends
- Higher win rate in trending markets
- Larger drawdowns in chop
- Medium frequency trading

Good for: Bull/bear markets
Bad for: Sideways markets
```

#### Mean Reversion
```
Characteristics:
- Fades extremes
- High win rate
- Small wins, occasional large loss
- High frequency typically

Good for: Range-bound markets
Bad for: Strong trends
```

#### Arbitrage
```
Characteristics:
- Market neutral
- Consistent small gains
- Low drawdowns
- High frequency

Good for: All markets
Risk: Technical failures
```

#### Momentum
```
Characteristics:
- Buys strength, sells weakness
- Lower win rate
- Large wins when right
- Medium frequency

Good for: Volatile markets
Bad for: Low volatility
```

## Investing in Vaults

### Portfolio Approach

#### Diversification Strategy
```
Don't put all in one vault
Spread across:
- Different strategies
- Multiple traders
- Various risk levels
- Different time horizons

Example allocation:
- 30% Conservative vault
- 40% Moderate vaults (2-3)
- 20% Aggressive vault
- 10% Experimental
```

#### Risk Management
```
Per vault limits:
- Maximum 20% in any vault
- Start with minimum investment
- Scale up gradually
- Monitor daily initially
```

### Selecting Vaults

#### Performance Analysis
```
Look for:
- Consistent returns
- Positive Sharpe (>1.0)
- Reasonable drawdowns (<20%)
```

#### Strategy Fit
